A CLI-Based Order & Weight Calculation Tool

PieceWeight is a Rust-based command-line application designed to calculate total weight and pricing for piece-based food orders.

It provides a structured and reliable method to automate calculations in small retail food environments.

üöÄ Problem Statement

In small bakeries, butcher shops, and similar businesses, orders are frequently calculated manually.

  • Counting individual pieces
  • Calculating total weight
  • Computing final price
  • Verifying totals manually

Manual calculation increases service time and introduces the risk of human error. PieceWeight automates this workflow using a configurable CLI system.

‚ú® Core Features

Instant Calculation
Enter item and quantity to receive total weight and price immediately.
Formatted Receipt Output
Clear terminal-based receipt with subtotal and totals.
Batch Processing
Process multiple orders using CSV input files.
Configurable Product Definitions
Product weights and prices are defined in a TOML configuration file.
Cross-Compilation
Supports Linux (x86_64 and ARM64) builds via GitHub Actions.

üìñ How It Works

  1. Edit the configuration file with product names, weights, and prices.
  2. Run the application in interactive mode.
  3. Enter item name and quantity.
  4. Receive calculated totals and receipt output.
