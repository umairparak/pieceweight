A CLI-Based Order & Weight Calculation Tool

PieceWeight is a Rust-based command-line application designed to calculate total weight and pricing for piece-based food orders.

The project was developed as part of the 24W-GBAC academic examination.

üöÄ Problem Statement

In small food businesses, orders are often taken manually. Staff must:

  • Count individual pieces
  • Calculate total weight
  • Compute final price
  • Double-check totals to avoid mistakes

Manual calculation increases the risk of human error and slows down service. PieceWeight automates this process.

Configurable Products
Product weights and pricing defined in a TOML configuration file.
Cross-Compilation

üìñ How It Works

  1. Edit the configuration file with product weights and prices.
  2. Run the application in interactive mode.
  3. Enter item name and quantity.
  4. Receive calculated totals and receipt output.
